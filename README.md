# Student Course Planner

## Product Vision

Student Course Planner helps students manage course registration by allowing them to add courses, search for courses, register students, and monitor course capacity through a simple console application.

## Main Features

- Add a course
- List all courses
- Search by course code
- Register students
- Display full courses

## How to Compile and Run

Compile:

javac *.java

Run:

java Main

## Example Output

=== Student Course Planner ===
1. Add Course
2. List Courses
3. Search Course
4. Register Student
5. Display Full Courses
6. Exit

## Suggested Database Design

### Course

### Student

### Registration
